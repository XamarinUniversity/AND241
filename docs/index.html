Before you begin

This is an Android class. To complete the coding exercises, you will need an Android development environment with either an emulator or a device to run the exercise on. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

This class covers the use and design of built-in and custom gestures, both continuous and discrete in Android. We will apply what we learn about gestures to create a multi-gesture application which includes a custom built rotation gesture as well as get hands-on experience designing discrete gestures using the Gestures Builder application, which is available on all Android devices.

Objectives

  1. Utilize built-in gestures
  2. Develop custom gestures programmatically
  3. Create discrete gestures using Gestures Builder

Lecture

Gestures

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Create an app that utilizes the scrolling gesture
    Exercise
    In this exercise, we will apply the standard gesture pattern to create an app that uses the scrolling pattern to move an image.
  2. Add the scaling gesture to your app
    Exercise
    In this exercise, we will modify our existing app and apply the scale gesture detector to create an app that can move and scale an image.
  3. Create a rotation gesture
    Exercise
    In this exercise, we will create a custom gesture by building our own gesture detector and add it to our existing app to enable it to scale, scroll and rotate an image.
